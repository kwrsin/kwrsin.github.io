1. Overview

The purpose of this application is to support document creation. You can store text fragments in this application and retrieve them when you need them. this application works best when working with other devices. Also, this function uses Apple's Universal Clipboard feature, so we do not support platforms other than Apple products such as iPad or Mac.

2. Features

2.1. Clipsboard
Top of the screen Displays the search criteria for these selected tags. Click here to specify the search conditions. If no search condition is specified, it will switch to the tag icon and you can set the search condition by clicking here.
List Displays the list of clips you have clipped. Tap to copy a clip. Tap and hold on a clip to delete it, tag it, title it, display it in the browser, or specify its color.
Bulk(Flag) you can tap the flag symbol at the bottom right to enter the bulk operation mode, where you can select multiple clips, tagging/untagging, and delete selected clips.

2.2. Settings: Specify options
Get a title from the URL When a clip is created, if the content is a URL, it will try to get the title from the URL.
give a cell color Adds a color option to the long-press context menu. - "show tags in cell": Displays the tags of the clip in the cell.
use internal browse you can tap the flag symbol at the bottom right to enter the bulk operation mode, where you can select multiple clips, tagging/untagging, and delete selected clips.
open browser on tap Open a URL by clip tapping if the clip data is URL.
Auto Tagging Automatically attach tags to new clips when they are registered.
  • Filtered
    • you can paste tags from the current search criteria.
  • Designated
    • you can specify tags individually and paste them.
Cache Display the cache size, and tap "Clear cache" to delete all clips.
Export you can export the clip data. If you select "All", you can export all clips to external applications. Select "All" to export all clips to external applications. If you select "Tags", you can select individual tags from the tag list and export them. Select "Tags" to select individual tags from the tag list and start exporting.

*Pastack file Importing

Use "Share" from an other app. From the application you are transferring from, select the Pastack file, Please find the Pastack app icon from "Share", and select that icon. Pastack app will automatically launch and start importing.

2.3. Search Conditions
Intersection Show all clips that match the selected tag's criteria.
Union Displays clips that match any of the selected tags.
Tags
  • Deselect All
    • you can deselects all selected tags.
  • Tag List
    • you can select the tags you want to search for.
History
  • History List
    • you can filter the clips by selecting the search criteria.
  • Favorites(heart)
    • you can save the search criteria items.

* You can save up to 5 history and 5 favorites.

+(Add) button you can register a new tag.

2.4. Press and Hold
Delete Deletes the clip.
Tagging Paste a tag to the clip
Title Give a title to the clip
Open the URL Opens an external browser with the clip's URL.
Copy the clip Copy a clip in clipsboard.
Color Add a color to the clip.
Transform Converts the clip data.
Share Open a Share Sheet.

2.5. Output
Processed
  • Tap
    • Copies the data in a cell.
  • Share (long press)
    • shares the data in a cell.
Processing
  • Cancel
    • cancel the process.
