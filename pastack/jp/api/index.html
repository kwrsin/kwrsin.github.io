パスタック Transformation APIs v1.8

概要

タグに属するクリップデータを別のデータに変換します。
下のような文法のクリップデータを登録したあと、長押しして、それを処理するために「変換を」選択します。

これらのAPIはクリップデータを処理するための命令です。

文法

//pastack

//タグに属するすべてのクリップを //-- と --// で囲みます
function _transform() {
  return getClips("Daily")
    .map(clip=>(`//-- ${clip.content} --//`))
}

Description

  • javascriptでコーディングします。
  • 先頭行 "//pastack" は「変換」を認識するのに必要です。
  • 必ず_transform() 関数の中でコードを書いてください。 処理結果に値を表示するために、String型要素を持つ配列を返却してください。
  • 処理中にエラーが発生した場合は、 処理結果にエラーが表示されます。
