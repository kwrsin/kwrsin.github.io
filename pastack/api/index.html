Pastack Transformation APIs v1.8

Overview

you can retrieves clip data of a tag and converts that data to other data.
After registering the following syntax as clip data, press and hold the clip and "transform" it.

These APIs are instructions for processing clip data.

Syntax

//pastack

//Enclose all clips belonging to a tag with //-- and --//
function _transform() {
  return getClips("Daily")
    .map(clip=>(`//-- ${clip.content} --//`))
}

Description

  • The code is written in javascript.
  • The first line "//pastack" is required to recognize the transformation.
  • Be sure to write the code to be processed in the _transform() function. Then return them as an array of strings to display those values in the output.
  • If errors occur during processing, the output will display the error details.
